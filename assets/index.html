Welcome to Let's Do

A new standard in managing your tasks on Android

Learn how to use the app

You can create new tasks

You can add descriptions to tasks

You can add sub-tasks

You can add pictures

Logging In

When opening the app for the first time, you'll be presented with a sign in screen

If you already have an account, simply sign in with your email and password

For new users, please tap register in the top right to create an account

Clicking either sign in or register will get you to the homepage of Let's Do!

Adding new tasks

From the home screen, look towards the bottom and tap the blue add button

You will now be prompted to type in a title for your new task

Tap 'enter' on your keyboard to save the title

You can now add optional information to your task, such as description

Remmeber to tap 'enter' on your keyboard to save the description

Adding sub-tasks

Whilst in an app, tap 'new sub-task...' and type the name of the sub-task

Tap 'enter' to save the sub-task and you'll see it appear with a checkbox below

You can tap an existing sub-task to mark it as complete

Notice that the sub-task greys out with a strikethrough when completed

View all tasks

All tasks can be viewed from the homescreen

You can see the title and description of each task

Scrolling down allows you to view more tasks if you have more than fit on the screen

Tap on any task to bring up more details and edit it

Editing a task

Title and description

Tap on the title or description to edit them

Simply make any changes you would like using th keyboard, and tap 'enter'

Sub-tasks

To complete a sub-task simply tap the checkbox or the text

The sub-task will grey out with strikethrough and the checkbox will fill

If a completed task is no longer complete, simply tap the checkbox or text again

Adding photos

To add a photo tap the camera button in the bottom left of your screen

This will bring up the camera view, tap the camera to take the photo, and tap the tick to confirm

The photo will not appear below the task description

Deleting a task

When the task is on the screen, look at the bottom right

Simply tap the red delete button to delete the task

The task, along with all associated sub-tasks and photos will be deleted

You can delete all tasks by tapping and holding on the home screen, then tap 'delete all'
